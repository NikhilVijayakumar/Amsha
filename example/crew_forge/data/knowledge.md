# Project Amsha Overview

Amsha 2.0 is a modular agentic framework designed for:
1.  **Flexibility**: Supporting multiple LLM providers (Gemini, LM Studio).
2.  **Scalability**: Using a robust specific Orchestrator.
3.  **Knowledge**: Integrating dynamic knowledge sources like this Markdown file.

## Key Components
-   **LLM Factory**: Manages provider connections.
-   **Crew Forge**: Builds and orchestrates agents.
-   **Execution Runtime**: Handles async task execution.
